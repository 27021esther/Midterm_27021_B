1. **You**: PRACTICAL EXAMINATION
DATABASE & SPRING BOOT APPLICATION DEVELOPMENT
Names: Patrick DUSHIMIMANA University: Adventist University of Central Africa Course: Web Technology and Internet Date: February 20, 2026
Instructions

Develop a Spring Boot project that demonstrates the required database relationships and functionalities. 2. You must clearly explain the logic behind each implementation. 3. Marks will be awarded according to the rubric below.
Practical Assessment Criteria (Total: 30 Marks)
Entity Relationship Diagram (ERD) with at least FIVE (5) tables. The student must explain the logic and relationships clearly. (3 Marks)
Implementation of saving Location. The student must explain how the data is stored and how relationships are handled. (2 Marks)
Implementation of Sorting functionality. The student must explain how sorting is implemented (e.g., using Pageable or Sort in Spring Data JPA). Implementation of Pagination. The student must explain how pagination works and how it improves performance. (5 Marks)
Implementation of Many-to-Many relationship. The student must explain the join table and how the relationship is mapped. (3 Marks)
Implementation of One-to-Many relationship. The student must explain the relationship mapping and foreign key usage. (2 Marks)
Implementation of One-to-One relationship. The student must explain how entities are connected. (2 Marks)
Implementation of existBy() method. The student must explain how existence checking works in Spring Data JPA. (2 Marks)
Retrieve all users from a given province using province code OR province name. The student must explain the query logic and repository method used. (4 Marks)
Viva-Voce Theory Questions (Student must explain clearly): (7 Marks) NB: Deadline is 13th
Marking Summary
Total Marks: 30
For these requirements do them on this topic vehicle trading management system ,A system that helps manage the buying and selling of vehicles by recording vehicle details, customer information, and transaction
